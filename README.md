This project is a **dynamic and interactive Sales Dashboard** created using the popular **Superstore dataset** in Power BI. It highlights key business insights such as sales trends, top-performing categories, regional performance, and profit margin analysis.

---

##  Features

- KPIs: Total Sales, Total Profit, Total Quantity, Profit Margin
- Trend analysis of sales over time
- Sales distribution by Region and State
- Category and Sub-Category performance
- Interactive slicers for Category, Region, Sub-Category, and Date

---

##  Files

- `Superstore_Sales_Dashboard.pbix` - Power BI report file
- `screenshots/dashboard.png` - Dashboard preview image

---

##  Key Insights

-  **South Region** leads in overall sales and profit
-  **Technology** category performs best across all metrics
-  Some sub-categories like **Tables** have low profit margin despite high sales
-  Sales peak during year-end seasons

---

##  Tools Used

- Power BI Desktop
- DAX for calculated measures (Profit Margin)
- Microsoft Excel (Superstore Dataset)
